## [1.0.0] - 2026-01-28

### Added
- Initial release of @your-scope/agent-kit
- Interactive CLI for project initialization
- Support for 9 project types (frontend, backend, mobile, fullstack, desktop, cli, extension, game, monorepo)
- Intelligent agent and skill filtering based on project type
- Configuration override system via `.agent-config.json`
- Update command to refresh existing configurations
- Comprehensive documentation and examples

### Features
- 20 specialist agents
- 36 skills
- 11 workflows
- Shared agents included in all project types
- Customizable agent/skill selection
- Preserves user configuration during updates
